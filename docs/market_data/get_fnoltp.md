To get the ltp of FnO, use the getFnoLtp method.

```python
Endpoint: /api/v1/marketdata/{exchange}/FutOpt?token={token}
Method-Type: GET
Method: pocket.getFnoLtp(Parameters)
```

## Path Parameters
| Field Name | Data Type | Description                           |
|------------|-----------|---------------------------------------|
| exchange   | String    | `NFO`                              |
| token      | String    | e.g. `26009`                            |
